# Guía Simple de Uso: App de Accidentes Escolares

## 1. ¿Qué es esta app?

Esta app sirve para registrar y ordenar accidentes escolares de forma simple y rápida.

En vez de usar planillas Excel o escribir todo a mano, aquí puedes:

- guardar accidentes
- buscar alumnos
- revisar historiales
- ver estadísticas
- generar reportes en PDF

Su objetivo es ayudar a que toda la información quede más ordenada, más clara y más fácil de encontrar.

## 2. ¿Para qué sirve?

La app te ayuda a:

- tener un registro digital de accidentes
- encontrar rápidamente la información de un alumno
- saber cuántos accidentes han ocurrido
- ver en qué cursos o lugares ocurren más accidentes
- mantener un historial ordenado
- imprimir reportes cuando sea necesario

## 3. Cómo entrar a la app

### Paso 1

Abre el enlace de la app en tu navegador.

### Paso 2

Escribe tu correo y tu contraseña.

### Paso 3

Presiona `Iniciar sesión`.

### Paso 4

Cuando termines de usar la app, puedes salir con el botón `Cerrar sesión`.

## 4. Qué significan los tipos de usuario

### Administrador

El administrador puede:

- crear
- editar
- eliminar
- administrar usuarios
- administrar lugares y tipos de lesión
- importar información

### Operador

El operador solo puede:

- revisar información
- consultar alumnos
- ver accidentes
- mirar el dashboard

El operador no puede crear, editar ni borrar datos.

## 5. Cómo moverse dentro de la app

En el menú lateral encontrarás estas secciones:

### Dashboard

Sirve para ver el resumen general.

### Alumnos

Sirve para buscar y revisar fichas de estudiantes.

### Accidentes

Sirve para ver el historial y registrar accidentes.

### Ajustes

Solo aparece para el administrador.  
Sirve para manejar usuarios, lugares, tipos de lesión e importaciones.

## 6. Cómo usar la sección Alumnos

### Para buscar un alumno

1. Entra a `Alumnos`
2. Busca por nombre, RUT o revisa el listado
3. Haz clic sobre el alumno que quieres revisar

### En la ficha del alumno podrás ver

- ID del alumno
- nombre
- RUT
- curso
- edad
- fecha de nacimiento
- apoderado
- teléfono
- salud / alergias
- observaciones

### Si eres administrador

También podrás:

- editar la ficha
- actualizar datos del alumno

### Si eres operador

Solo podrás mirar la información.

## 7. Cómo usar la sección Accidentes

### Para ver el historial

1. Entra a `Accidentes`
2. Revisa la lista de accidentes registrados
3. Puedes filtrar por datos como RUT, curso, lesión o fecha

### Para registrar un accidente

1. Entra a `Accidentes`
2. Presiona `Registrar accidente`
3. Escribe el RUT del alumno
4. La app completará automáticamente los datos del alumno si está en la base
5. Completa los datos del accidente
6. Presiona `Guardar accidente`

### Qué datos puedes registrar

- fecha
- hora
- lugar
- tipo de lesión
- descripción
- primeros auxilios
- derivación
- centro de salud
- aviso al apoderado
- responsable
- observaciones

### Para abrir el detalle de un accidente

1. Entra al historial
2. Haz clic sobre el número de registro

### En la ficha del accidente podrás ver

- datos del alumno
- fecha y hora
- lugar
- lesión
- descripción
- observaciones
- responsable
- si hubo derivación
- si se informó al apoderado

### Si eres administrador

También podrás:

- editar el accidente
- guardar cambios
- eliminar el accidente
- abrir el PDF del accidente

### Si eres operador

Solo podrás revisar la información.

## 8. Cómo usar el Dashboard

El `Dashboard` sirve para mirar el resumen general de la información.

Ahí puedes ver:

- total de accidentes
- accidentes con derivación
- accidentes sin aviso al apoderado
- accidentes sin descripción
- último registro
- curso con más accidentes
- lesión más frecuente
- día con más accidentes

También puedes ver tablas como:

- accidentes por curso
- accidentes por tipo de lesión
- accidentes por día

### Si necesitas un reporte

Puedes abrir el PDF del resumen desde el mismo dashboard.

## 9. Cómo usar Ajustes

Esta sección es solo para el administrador.

### Aquí puedes:

- crear usuarios
- editar usuarios
- eliminar usuarios
- agregar lugares
- editar lugares
- eliminar lugares
- agregar tipos de lesión
- editar tipos de lesión
- eliminar tipos de lesión
- importar alumnos
- importar accidentes

## 10. Cómo importar información

Si tienes información en un archivo, el administrador puede importarla desde `Ajustes`.

### Se puede importar

- alumnos
- accidentes

### Antes de importar

Revisa que el archivo sea el correcto y que la información esté ordenada.

## 11. Recomendaciones simples de uso

- revisa siempre los datos antes de guardar
- usa el RUT para encontrar alumnos más rápido
- si eres administrador, revisa bien antes de eliminar un registro
- usa el historial para confirmar que un accidente sí quedó guardado
- si un cambio no aparece de inmediato, presiona `Ctrl + F5`

## 12. Resumen final

Esta app fue hecha para que cualquier persona pueda:

- registrar accidentes escolares
- revisar alumnos
- consultar historiales
- ver estadísticas
- trabajar de forma más ordenada y más rápida

Si eres administrador, tendrás control total.  
Si eres operador, podrás revisar la información de forma segura sin modificarla.
