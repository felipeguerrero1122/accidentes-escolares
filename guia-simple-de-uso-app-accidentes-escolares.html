Guia Simple de Uso

App de Accidentes Escolares

Esta app sirve para registrar y ordenar accidentes escolares de forma simple, rapida y clara.

En vez de usar planillas Excel o escribir todo a mano, aqui puedes guardar accidentes, buscar alumnos, revisar historiales, ver estadisticas y generar reportes en PDF.

1. Para que sirve

  • Tener un registro digital de accidentes.
  • Encontrar rapidamente la informacion de un alumno.
  • Ver cuantos accidentes han ocurrido.
  • Revisar en que cursos o lugares ocurren mas accidentes.
  • Mantener un historial ordenado.
  • Imprimir reportes cuando sea necesario.

2. Como entrar a la app

  1. Abre el enlace de la app en tu navegador.
  2. Escribe tu correo y tu contraseña.
  3. Presiona Iniciar sesion.
  4. Cuando termines, presiona Cerrar sesion.

3. Tipos de usuario

Administrador

  • Puede crear.
  • Puede editar.
  • Puede eliminar.
  • Puede administrar usuarios.
  • Puede administrar lugares y tipos de lesion.
  • Puede importar informacion.

Operador

  • Solo puede revisar informacion.
  • Puede consultar alumnos.
  • Puede ver accidentes.
  • Puede mirar el dashboard.
  • No puede crear, editar ni borrar.

4. Como moverse dentro de la app

  • Dashboard: muestra el resumen general.
  • Alumnos: permite buscar y revisar fichas de estudiantes.
  • Accidentes: permite ver el historial y registrar accidentes.
  • Ajustes: solo para administrador; sirve para usuarios, lugares, tipos de lesion e importaciones.

5. Como usar Alumnos

Para buscar un alumno

  1. Entra a Alumnos.
  2. Busca por nombre, RUT o revisa el listado.
  3. Haz clic sobre el alumno que quieres revisar.

En la ficha del alumno puedes ver

  • ID del alumno
  • Nombre
  • RUT
  • Curso
  • Edad
  • Fecha de nacimiento
  • Apoderado
  • Telefono
  • Salud o alergias
  • Observaciones

Si eres administrador, tambien puedes editar la ficha. Si eres operador, solo puedes mirar la informacion.

6. Como usar Accidentes

Para ver el historial

  1. Entra a Accidentes.
  2. Revisa la lista de accidentes registrados.
  3. Si quieres, filtra por RUT, curso, lesion o fecha.

Para registrar un accidente

  1. Entra a Accidentes.
  2. Presiona Registrar accidente.
  3. Escribe el RUT del alumno.
  4. La app completara automaticamente sus datos si ya esta en la base.
  5. Completa la informacion del accidente.
  6. Presiona Guardar accidente.

Que datos puedes registrar

  • Fecha
  • Hora
  • Lugar
  • Tipo de lesion
  • Descripcion
  • Primeros auxilios
  • Derivacion
  • Centro de salud
  • Aviso al apoderado
  • Responsable
  • Observaciones

Para abrir el detalle de un accidente

  1. Entra al historial.
  2. Haz clic sobre el numero de registro.

Si eres administrador, tambien puedes editar el accidente, guardar cambios, eliminarlo y abrir el PDF. Si eres operador, solo puedes revisar la informacion.

7. Como usar el Dashboard

El dashboard sirve para mirar el resumen general de la informacion.

  • Total de accidentes
  • Accidentes con derivacion
  • Accidentes sin aviso al apoderado
  • Accidentes sin descripcion
  • Ultimo registro
  • Curso con mas accidentes
  • Lesion mas frecuente
  • Dia con mas accidentes

Tambien puedes revisar tablas de accidentes por curso, por tipo de lesion y por dia. Si lo necesitas, puedes abrir el PDF del resumen.

8. Como usar Ajustes

Esta seccion es solo para el administrador.

  • Crear usuarios
  • Editar usuarios
  • Eliminar usuarios
  • Agregar, editar y borrar lugares
  • Agregar, editar y borrar tipos de lesion
  • Importar alumnos
  • Importar accidentes

9. Recomendaciones simples

  • Revisa siempre los datos antes de guardar.
  • Usa el RUT para encontrar alumnos mas rapido.
  • Si eres administrador, revisa bien antes de eliminar un registro.
  • Usa el historial para confirmar que un accidente si quedo guardado.
  • Si un cambio no aparece de inmediato, presiona Ctrl + F5.

Resumen final: esta app fue hecha para que cualquier persona pueda registrar accidentes escolares, revisar alumnos, consultar historiales, ver estadisticas y trabajar de forma mas ordenada.

Administrador: control total. Operador: consulta segura sin modificar informacion.
