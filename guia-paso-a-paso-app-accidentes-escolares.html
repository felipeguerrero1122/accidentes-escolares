Guia paso a paso

App de Accidentes Escolares

Esta guia fue hecha para cualquier persona. Su objetivo es mostrar de forma simple como entrar a la app, como buscar alumnos, como registrar accidentes y como revisar la informacion sin complicaciones.

1. Como entrar

Primero abre el enlace de la app. Luego escribe tu correo y tu contraseña. Despues presiona Ingresar.

Pantalla de ingreso.

  1. Escribe tu correo.
  2. Escribe tu contraseña.
  3. Presiona Ingresar.
  4. Cuando termines, usa Cerrar sesion.

2. Que puedes hacer segun tu tipo de usuario

Administrador

  • Puede crear registros.
  • Puede editar alumnos y accidentes.
  • Puede eliminar accidentes, usuarios y catalogos.
  • Puede entrar a Ajustes.

Operador

  • Puede revisar la informacion.
  • Puede consultar alumnos y accidentes.
  • Puede mirar el dashboard.
  • No puede crear, editar ni borrar.

3. Dashboard

El dashboard sirve para ver el resumen general de la informacion. Muestra cantidades, cursos con mas accidentes, lesiones frecuentes y tablas por fecha, curso o lesion.

  1. Revisa los cuadros de arriba para ver el resumen rapido.
  2. Mira las tablas para entender donde ocurren mas accidentes.
  3. Si lo necesitas, abre el PDF del resumen.

4. Alumnos

Listado de alumnos

En esta pantalla puedes buscar por nombre, RUT o curso.

  1. Entra a Alumnos.
  2. Busca por nombre, RUT o codigo.
  3. Haz clic sobre el alumno para abrir su ficha.

Ficha del alumno

Aqui puedes ver los datos principales del estudiante.

  • Nombre
  • RUT
  • Curso
  • Edad
  • Fecha de nacimiento
  • Apoderado
  • Telefono

Si eres administrador, tambien puedes editar la ficha.

5. Accidentes

Historial de accidentes

Esta pantalla te muestra todos los registros guardados.

  1. Entra a Accidentes.
  2. Revisa los registros guardados.
  3. Usa los filtros si quieres buscar algo especifico.
  4. Haz clic en un numero de registro para abrir el detalle.

Nuevo accidente

Esta es la pantalla para registrar un accidente nuevo.

  1. Escribe el RUT del alumno.
  2. La app completara automaticamente sus datos si ya esta en la base.
  3. Completa fecha, hora, lugar y lesion.
  4. Escribe la descripcion del accidente.
  5. Presiona Guardar accidente.

Detalle del accidente

En esta ficha puedes revisar toda la informacion del registro.

  • Datos del alumno
  • Fecha y hora
  • Lugar
  • Tipo de lesion
  • Descripcion
  • Observaciones

Si eres administrador, desde aqui tambien puedes editar el accidente, guardar cambios, abrir el PDF o eliminarlo.

6. Ajustes

Lugares, tipos de lesion e importaciones

Esta parte es solo para el administrador.

  • Agregar nuevos lugares
  • Agregar nuevos tipos de lesion
  • Editar o borrar lugares
  • Editar o borrar tipos de lesion
  • Importar alumnos o accidentes desde archivo

Usuarios

El administrador puede manejar quienes entran a la app.

  • Crear usuarios
  • Editar nombre, correo o rol
  • Eliminar usuarios

7. Consejos simples

  • Revisa los datos antes de guardar.
  • Usa el RUT para encontrar alumnos mas rapido.
  • Si un cambio no aparece, presiona Ctrl + F5.
  • Si eres administrador, revisa bien antes de borrar informacion.

8. Resumen final

Esta app permite registrar accidentes escolares, revisar alumnos, consultar historiales y ver estadisticas de forma ordenada y facil de entender.

Administrador: control total. Operador: consulta segura sin modificar informacion.
